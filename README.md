
# AI Loan Approval System Analysis

This project presents a detailed analysis of an AI-based loan approval system using machine learning models.

## Description
The analysis evaluates the performance of two models:
- Random Forest
- Deep Learning (Neural Network)

The models were trained on a dataset of 5,000 records and tested on new loan applications.

## Results
- Random Forest achieved higher accuracy compared to the neural network
- The system selects top candidates based on prediction probability
- The analysis shows how income and loan amount affect approval decisions

## Features
- Model performance evaluation
- Comparison between AI models
- Risk assessment based on financial data

## Technologies Used
- Python
- Machine Learning
- Data Analysis
